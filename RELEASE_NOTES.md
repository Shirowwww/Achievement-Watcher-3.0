# Achievement Watcher 3.2.1

Version 3.2.1 is a focused refinement of the first-run guide, making the initial setup clearer, more responsive, and easier to get right.

## Highlights

- See your progress through setup with a step counter, a progress bar, and completed-step markers.
- Validate a Steam Web API key as you type, with live valid/malformed feedback and automatic paste cleanup, plus a show/hide control.
- Send a test notification straight from the guide using the transport you just picked (toast, overlay, or both).
- Reopen the guide from Settings and dismiss it reliably, including with the keyboard.
- Keep the guide usable at the minimum window height.

See the [changelog](CHANGELOG.md#321---2026-07-14) for the full list of changes.

## Install

Download `Achievement.Watcher.Setup.3.2.1.exe` from the [v3.2.1 release](https://github.com/Shirowwww/Achievement-Watcher-3.0/releases/tag/v3.2.1).

The `.blockmap` and `latest.yml` assets are used by automatic updates. Existing settings and tracked data under `%APPDATA%\Achievement Watcher` are preserved when upgrading.
